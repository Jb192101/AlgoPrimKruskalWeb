# AlgoPrimKruskalWeb
Репозиторий с помещением алгоритма Прима-Краскала в веб-оболочку с помощью Java EE.

## Технологии:
- Java Core;
- Java EE;
- Apache Tomcat.

Реализация алгоритма находится в [этом репозитории](https://github.com/Jb192101/AlgorithmPrimaKraskala)
